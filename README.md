Project Overview

This project is developed as part of a Python programming assignment. It demonstrates fundamental Python concepts, including variables, data types, reserved keywords, and arithmetic operations. The project helps beginners understand Python syntax and basic programming logic through simple examples.

Objectives

- Create and use variables in Python.
- Check the data type of a variable using the "type()" function.
- Understand why reserved keywords cannot be used as variable names.
- Calculate Simple Interest using Python variables and arithmetic operators.

Features

- Variable creation and initialization.
- Data type identification.
- Demonstration of Python reserved keywords.
- Simple Interest calculation using the standard formula.
